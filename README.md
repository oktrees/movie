# INFO

* 책 커버 디자인 사이트입니다</br>
* 관리자 로그인 기능
* 관리자는 BOOKLIST 탭 게시판에 작업물을 게시, 수정, 삭제, 이미지 업로드를 할 수 있습니다.<br/> 
* 관리자는 CONTACT 탭에서 회사 소개 글을 수정할 수 있습니다. <br/>
* 게시한 작업물들은 메인페이지에 carousel로 노출됩니다.<br/>
